== Description ==

Remove the Toolbar (a.k.a Admin Bar) completely from both front and back end.

= Plugin Features =

1.Disables the wp toolbar (or) admin bar in the **dashboard**.

2.Disables the toolbar in the **front end**.

3.Removes the unnecessary **action hooks**.

4.Makes the necessary **css tweaks**.

5.Minimizes **memory load** on wordpress.

= Works with Wordpress 3.3.1 =
Also works with the **Ozh Admin Drop Down Menu**

Installing this plugin not only disables the toolbar in the front end but also removes it completely from the backend. All the *action hooks* will be removed, thus optimizing the page load time.

== Installation ==

1.Install through the WordPress admin or upload the plugin folder to your /wp-content/plugins/ directory.

2.Activate the plugin through the ‘Plugins’ menu in WordPress.
