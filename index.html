Cj-remove-toolbar

Remove the Toolbar(Admin Bar) completely from both front end and back end.

View project onGitHub

The Admin bar is transformed into a Toolbar tucked neatly above the Admin area in the latest version of WordPress. Some people love it, some don’t.

If you don’t like the new Toolbar, then you will LOVE this plugin!

Plugin Name:

CJ Remove WP Toolbar

Description:

Remove the Toolbar(Admin Bar) completely from both front end and back end.

WordPress 3.3.1 compatible.

All the action hooks will be removed, thus optimizing the page load time.

Version:

3.3.1
